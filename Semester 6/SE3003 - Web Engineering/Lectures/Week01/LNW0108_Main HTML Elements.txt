Main HTML Elements
==================
* BODY Element
<BODY BGCOLOR="YELLOW">
	HTML Attributes and Attribute Values
	BACKGROUND
	BGCOLOR
	TEXT
	LINK, VLINK, ALINK
	OnLoad, OnUnload, OnFocus, OnBlur
* Elements inside BODY element
<BODY>
	Remaining HTML elements
</BODY>

Example:
<BODY onBlur="alert('Body onBlur alert!');"
onError="alert('Body onError alert!');"
onFocus="alert('Body onFocus alert!');"
onLoad="alert('Body onLoad alert!');"
onUnload="alert('Body onUnload alert!');">

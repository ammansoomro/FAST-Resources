Main HTML Elements
==================

* DOCTYPE
* HTML
* HEAD
	TITLE element required 
	Optional elements: 
		BASE 
		META 
		BGSOUND
		SCRIPT, NOSCRIPT
		STYLE 
		LINK 


* The <link> tag is used to link to external style sheets.
<link rel="stylesheet" type="text/css" href="theme.css">

Style tag for inline html style:
<style> h1 {color:red;} p {color:blue;} </style>
